🌐 Network Access

Host Options:
• 0.0.0.0 - Accessible from any device on your network
• 127.0.0.1 - Only accessible from this computer
⚠️ Security: Using 0.0.0.0 allows network access. Make sure you trust devices on your network or use IP whitelist below.

🔌 Port Configuration

Common Ports:
• 80 - Standard HTTP (requires admin/root)
• 8080 - Alternative HTTP (no special privileges)
• 5000 - Common development port
• 3000 - Alternative development port
⚠️ Note: Port 80 requires administrator/root privileges.

⚡ Performance Settings

Update Interval:
• 0.1-0.5s - Very responsive, higher CPU usage
• 0.5-1.0s - Balanced (recommended)
• 1.0-5.0s - Lower CPU usage, less responsive
💡 Tip: Lower values = more responsive but use more system resources.

🔒 Security Settings

IP Whitelist Examples:
• 127.0.0.1 or ::1 - localhost only
• 192.168.1.100 - specific IP
• 192.168.2.0/24 - entire subnet
• 0.0.0.0/0 - allow all (not recommended)
⚠️ Only whitelisted IPs can access settings pages. Index page is always accessible.

Python strftime Format Codes:
• %Y - 4-digit year (2025)
• %m - 2-digit month (01-12)
• %d - 2-digit day (01-31)
• %H - 2-digit hour 24h (00-23)
• %M - 2-digit minute (00-59)
• %S - 2-digit second (00-59)
• %A - Weekday name (Monday)
• %B - Month name (January)
• %W - Week number (00-53)
Path Format Examples:
• %Y/%m → 2025/01/
• %Y/%m/%d → 2025/01/08/
• Daily/%Y-%m-%d → Daily/2025-01-08/
• Weekly/%Y-W%W → Weekly/2025-W02/
• %B_%Y → January_2025/
Filename Format Examples:
• %Y-%m-%d_%H%M%S → 2025-01-08_143022
• %Y%m%d_%H%M → 20250108_1430
• %A_%Y-%m-%d → Wednesday_2025-01-08

🎙️ Audio Backup

Two backup formats available:
• .wav - Final session file, created when transcription stops
• .ts - Continuous MPEG-TS backup, power-fail safe (survives crashes)
Backup Settings:
• Both formats use the same path and filename settings
• Default base path: _AUTOMATIC_BACKUP
• Uses Python strftime format codes for paths and filenames
Audio Path Examples:
• %Y/%m → 2025/01/
• ByMonth/%Y-%m → ByMonth/2025-01/
• Daily/%Y-%m-%d → Daily/2025-01-08/
Audio Filename Examples:
• %Y-%m-%d-%H%M%S → 2025-01-08-143022.wav/.ts
• Voice_%Y-%m-%d → Voice_2025-01-08.wav/.ts
⚠️ Audio backups can consume significant disk space over time. The .ts format is recommended for long recordings as it survives power failures.
